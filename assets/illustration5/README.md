# Illustration 5 - HEALTH Category

This folder should contain a `textures.zip` file with tile images for the HEALTH category.

The zip file should contain PNG files named in the format: `tile_row_col.png` (e.g., `tile_0_0.png`, `tile_1_5.png`)

Grid dimensions: 50 rows × 62 columns
Tile size: 20px × 20px

This corresponds to the **BottomLeft** hexagonal direction (120° to 180°).
